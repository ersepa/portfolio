    Hosting

    Hosting adalah layanan online yang berfungsi menyimpan data dan file website (seperti HTML, gambar, video, dan database) agar bisa diakses melalui internet. Ibaratnya, hosting itu rumahnya website, tempat semua file disimpan biar orang lain bisa buka lewat browser.

    Hosting memiliki beberapa jenis, yaitu: Shared hosting, VPS hosting, Cloud hosting, Dedicated hosting dan Free hosting. Masing-masing dari jenis hosting tersebut memiliki harga yang berbeda dan semakin mahal harganya akan lebih bagus spesifikasi server hosting nya.
